# Credit Card Fraud Detection SQL Analysis

## Project Overview
This project analyzes simulated credit card transaction data using SQL to identify suspicious spending behavior, declined transaction patterns, and potential fraud indicators.

The project was designed to demonstrate practical SQL skills relevant to financial analytics, fraud monitoring, and transaction risk analysis.

---

## Business Problem
Financial institutions must monitor millions of transactions daily to identify unusual activity and reduce fraud risk.

This project explores:
- High-dollar transactions
- Declined payment behavior
- Merchant fraud exposure
- Customer transaction trends

---

## Database Schema
The database contains five relational tables:

- customers
- accounts
- merchants
- transactions
- fraud_flags

The schema uses primary and foreign key relationships to model real-world banking transaction systems.
