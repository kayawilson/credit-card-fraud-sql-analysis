# Dashboard Plan

## Dashboard Goal
Create a visual summary of credit card transaction activity, fraud indicators, and account risk patterns.

## Planned Visuals
1. Approved vs Declined Transactions
2. Fraud Flags by Merchant
3. Total Spend by Merchant Category
4. High-Dollar Transactions Over Time

## Key Metrics
- Total transactions
- Total transaction amount
- Decline rate
- Number of fraud flags
- Average transaction amount

## Business Use Case
This dashboard would help analysts quickly identify suspicious transaction trends and monitor potential fraud risk across customers, accounts, and merchants.
